# Reproducibility Check Skill

Use this skill when the user wants to verify that their empirical thesis project can be reproduced.

## Checklist

Check whether the project has:

1. Raw data folder.
2. Clean data folder.
3. Code scripts with clear order.
4. Output folder for tables and figures.
5. README explaining how to run the project.
6. Requirements or environment file.
7. Data dictionary.
8. Clear distinction between raw and generated files.
