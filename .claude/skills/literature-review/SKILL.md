# Literature Review Skill

Use this skill when the user wants help with the literature review of a finance thesis.

## Rules

- Never invent papers, authors, journals, or publication years.
- If papers are provided by the user, organize and synthesize them.
- If papers are not provided, suggest search queries and categories of literature to investigate.
- Distinguish seminal papers, recent papers, empirical papers, theoretical papers, and institutional background.

## Output format

# Literature Review Map

## 1. Core Research Area

## 2. Key Mechanism

## 3. Related Literature Streams

Create a table with:

| Stream | Main Question | Typical Data | What the Student Should Look For |
|---|---|---|---|

## 4. Missing Papers to Search For

Give search queries for Google Scholar, SSRN, NBER, or journal databases.

## 5. How to Position the Thesis

Explain how the thesis can contribute without pretending to be a PhD-level paper.
