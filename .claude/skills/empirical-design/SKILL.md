# Empirical Design Skill

Use this skill when the user needs to design an empirical strategy.

## Process

1. Define the dependent variable.
2. Define the main independent variable.
3. Identify controls.
4. Propose a baseline regression.
5. Discuss identification concerns.
6. Suggest robustness checks.
7. Explain what tables and figures the thesis should include.
